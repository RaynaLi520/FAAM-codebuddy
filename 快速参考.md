# FAAM 快速参考

## 常用命令

### 启动Web应用
```bash
cd D:\CodeBuddy\FAAM
python app.py
```
访问地址: http://localhost:5000

### 安装依赖
```bash
pip install -r requirements.txt
```

### 导入数据
```bash
python import_data.py
```

### 导入历史数据
```bash
python import_history.py
```

### 运行自动化任务
```bash
python auto_scheduler.py
```

| 文件 | 说明 |
|------|------|

## 目录结构

| 目录 | 说明 |

## 批处理脚本

### start.bat - 启动Web应用
1. 检查Python环境
2. 检查并安装依赖
3. 初始化数据库
4. 启动Flask应用

### setup_task.bat - 安装定时任务（需管理员）
1. 检查Python环境
2. 验证脚本文件
3. 删除旧任务（如有）
4. 创建新的Windows任务计划
5. 设置每天18:30自动执行

### run_now.bat - 立即执行自动化任务
执行完整的数据爬取和导入流程

### import_history.bat - 导入历史数据
批量导入data目录下的所有Excel文件

## 数据库表

### products 表
存储所有商品信息

主要字段:
- tcin: 商品唯一标识
- title: 商品名称
- brand: 品牌
- price: 价格
- is_new: 是否新品
- image_url: 图片链接
- product_url: 购买链接

### daily_new_arrivals 表
记录每日发现的新品

主要字段:
- tcin: 商品标识
- title: 商品名称
- brand: 品牌
- price: 价格
- date_detected: 发现日期

## API接口

### GET /
首页

### GET /products
商品查询页面

### GET /new-arrivals
每日新品页面

### GET /api/products
获取商品列表JSON

参数:
- brand: 品牌筛选
- is_new: 是否新品 (yes/no)
- search: 搜索关键词

### GET /api/dashboard
获取仪表盘数据

参数:
- date: 日期筛选

## 任务计划管理

### 查看任务状态
```bash
schtasks /query /tn "FAAM_AutoCrawler"
```

### 禁用任务
```bash
schtasks /change /tn "FAAM_AutoCrawler" /disable
```

### 启用任务
```bash
schtasks /change /tn "FAAM_AutoCrawler" /enable
```

### 删除任务
```bash
schtasks /delete /tn "FAAM_AutoCrawler" /f
```

### 手动运行任务
```bash
schtasks /run /tn "FAAM_AutoCrawler"
```

### 更改执行时间
```bash
schtasks /change /tn "FAAM_AutoCrawler" /st 19:00
```

## 常见问题排查

| 问题 | 解决方案 |
|------|------|
| 找不到Flask模块 | 运行 `pip install Flask pandas openpyxl` |
| 数据库为空 | 运行 `python import_data.py` 导入数据 |
| 图片无法显示 | 检查商品数据中图片URL是否有效 |
| 定时任务不执行 | 检查任务是否启用，查看日志文件 |
| 爬虫失败 | 检查网络连接，确认Target网站可访问 |
